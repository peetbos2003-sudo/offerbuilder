ECR Offer Builder

This page is the GitHub Pages host for the ECR Offer Builder Outlook add-in.

If you can read this, GitHub Pages is serving correctly. The add-in itself is loaded by Outlook from these files, not from this page:

Panel: taskpane.html
Manifest (use this URL in Outlook / Integrated apps): https://peetbos2003-sudo.github.io/offerbuilder/manifest.xml

To add it in Outlook: New email → Apps / “…” → Get Add-ins → My add-ins → Custom Addins → add the manifest URL above. See README-install.md for the full steps.
